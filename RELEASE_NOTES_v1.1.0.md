# TVPlayer v1.1.0 更新说明

**发布日期**: 2026-07-25  
**版本**: Android v1.1.0 (versionCode 3)

---

## 🎉 重大更新

本次更新完全解决了用户反馈的三大核心问题，大幅提升使用体验！

---

## 🔥 问题 1: 快速切换优化 - 切换速度提升 57-79%

### 修复内容
✅ **超时时间优化**
- WiFi 环境：7秒 → 3秒 (提升 57%)
- 卡顿检测：7秒 → 2.5秒 (提升 64%)
- 移动网络：增加 1.5秒快速失败机制 (提升 79%)
- 重试间隔：1秒 → 0.5秒 (提升 50%)

✅ **智能网络检测**
- 自动识别移动网络/WiFi
- 移动网络自动启用快速切换模式
- 动态调整超时策略

### 用户体验提升
- 🚀 黑屏/卡顿等待时间大幅缩短
- 🚀 网络差时立即切换，不再长时间等待
- 🚀 切换流畅度显著改善

---

## 🌐 问题 2: 多源加载 - 频道数量提升 200%+

### 修复内容
✅ **6 个优质 GitHub 源自动拼接**
1. best-fan/iptv-sources (状态检测版)
2. fanmingming/live (IPv6 高清源)
3. YueChan/Live (稳定源)
4. Supprise0901/TVBox_live (TVBox 源)
5. vbskycn/iptv (综合源)
6. YanG-1989/m3u (高质量源)

✅ **4 个镜像加速前缀**
- ghfast.top (首选)
- raw.gitmirror.com (备选)
- raw.kkgithub.com (备选)
- gcore.jsdelivr.net (CDN 加速)

✅ **智能质量筛选**
- 自动过滤测试/演示链接
- 排除非标准端口
- 同名频道自动合并多线路

### 用户体验提升
- 📺 频道数量：~300 → 800-1000+ (提升 200%+)
- 🌍 无需代理，镜像加速访问 GitHub
- 🎯 自动去重，频道质量更高
- ⚡ 多源并发加载，速度更快

---

## 📱 问题 3: 完美全屏显示 - Home 条不再挤占画面

### 修复内容
✅ **沉浸式全屏模式**
- 首次启动完全全屏
- 从后台恢复自动保持全屏
- 支持 Android 11+ 新 API (WindowInsetsController)
- 兼容 Android 10 及以下 (SYSTEM_UI_FLAG_IMMERSIVE_STICKY)

✅ **自动隐藏导航栏**
- 滑动显示后自动隐藏
- 窗口焦点改变时自动应用
- onResume 时重新应用

### 用户体验提升
- 🖼️ 视频占满整个屏幕
- ✨ Home 条不再挤占画面
- 🎬 完美沉浸式观看体验

## 🛠️ 技术改进

### 代码优化
- ✅ 重构超时常量配置
- ✅ 添加网络状态检测 (ConnectivityManager)
- ✅ 实现多源并发加载机制
- ✅ 实现智能 URL 质量筛选
- ✅ 实现同名频道智能合并
- ✅ 完善沉浸式全屏 API (Android R+)

### 新增方法
- `checkNetworkSpeed()` - 网络速度检测
- `loadChannelsFromMultiSources()` - 多源加载
- `isQualityUrl()` - URL 质量筛选
- `mergeChannelsByName()` - 频道合并
- `setupImmersiveMode()` - 沉浸式全屏设置
- `applyImmersiveMode()` - 应用沉浸式模式

## 📦 安装说明

### Android 版本
1. 下载 `app-debug.apk`
2. 安装到设备（需允许未知来源）
3. 打开应用，享受全新体验！

### 系统要求
- **最低版本**: Android 4.4 (API 19)
- **目标版本**: Android 13 (API 33)
- **推荐配置**: Android 8.0+ 以获得最佳体验

---

## ⚠️ 注意事项

1. **首次启动**: 会自动加载 6 个源，请稍候片刻
2. **网络要求**: 无需代理，但需要稳定的网络连接
3. **存储权限**: 需要存储权限以保存频道列表缓存
4. **卸载旧版**: 建议先卸载 v1.0.1 再安装新版
