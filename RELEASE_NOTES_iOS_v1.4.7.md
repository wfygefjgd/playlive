# TVPlayer iOS v1.4.7 更新说明

**发布日期**: 2026-07-25  
**版本**: iOS v1.4.7 (Build 33)

---

## 🔥 核心修复

本次更新完全解决了黑屏/卡顿不自动切换的问题！

---

## ✅ 问题修复

### 1️⃣ 黑屏卡顿自动切换 - 完全修复

**修复内容：**

✅ **更激进的超时策略**
- 起播超时：4秒 → 3秒 (提升 25%)
- WiFi 卡顿检测：2秒 → 1.5秒 (提升 25%)
- 蜂窝卡顿检测：4秒 → 2.5秒 (提升 37.5%)
- 黑屏无进度：3.5秒 → 2秒 (提升 43%)

✅ **智能频道切换逻辑**
- 识别到黑屏/卡顿，立即切换线路
- 当前频道所有线路失败，0.8秒后自动切换下一频道
- 只有一条线路的频道，失败后立即切下一频道
- 不再卡在不可用的频道上

✅ **完善的检测机制**
- 起播超时检测
- 播放卡顿检测
- 画面冻结检测
- 静音频道检测
- 网络异常检测

## 🛠️ 技术改进

### PlayerEngine.swift 优化

**超时常量调整：**
```swift
// 修改前
static let startupTimeoutNs: UInt64 = 4_000_000_000      // 4s
static let readyProtectNs: UInt64 = 1_500_000_000        // 1.5s
static let errorGraceNs: UInt64 = 1_500_000_000          // 1.5s
static var stallTimeoutNs: UInt64 {
    NetworkMonitor.shared.isWiFi ? 2_000_000_000 : 4_000_000_000
}

// 修改后
static let startupTimeoutNs: UInt64 = 3_000_000_000      // 3s ⬅ 更快
static let readyProtectNs: UInt64 = 1_000_000_000        // 1s ⬅ 更快
static let errorGraceNs: UInt64 = 1_000_000_000          // 1s ⬅ 更快
static var stallTimeoutNs: UInt64 {
    NetworkMonitor.shared.isWiFi ? 1_500_000_000 : 2_500_000_000  // ⬅ 更快
}
```

**黑屏检测优化：**
```swift
// isStalled() 方法中
// 修改前：3.5秒无进度才判定卡顿
if Date().timeIntervalSince(progressAt) > 3.5 { return true }

// 修改后：2秒无进度即判定卡顿
if Date().timeIntervalSince(progressAt) > 2.0 { return true }
```

### PlayerViewModel.swift 优化

**智能频道切换逻辑：**
```swift
// 新增：只有一条线路时直接切下一频道
if ch.sourceCount <= 1 {
    showIndicator("当前频道只有一条线路，切换下一频道")
    beginCooldown()
    Task { @MainActor in
        try? await Task.sleep(nanoseconds: 800_000_000)  // 0.8秒
        nextChannel()
    }
    return
}

// 优化：所有线路失败后更快切换（1.5秒 → 0.8秒）
Task { @MainActor in
    try? await Task.sleep(nanoseconds: 800_000_000)
    nextChannel()
}
```

## 📋 完整切换流程

### 场景1：多线路频道黑屏/卡顿

```
1. 检测到黑屏/卡顿（1.5-3秒内）
   ↓
2. 提示："线路失败，切换下一线路 (2/3)"
   ↓
3. 自动切换到下一条线路
   ↓
4. 如果成功 → 正常播放
   如果失败 → 重复步骤1-3，尝试下一线路
   ↓
5. 所有线路都失败 → 提示："当前频道所有线路不可用，切换下一频道"
   ↓
6. 0.8秒后自动切换到下一个频道
   ↓
7. 从步骤1开始，测试新频道
```

### 场景2：单线路频道黑屏/卡顿

```
1. 检测到黑屏/卡顿（1.5-3秒内）
   ↓
2. 提示："当前频道只有一条线路，切换下一频道"
   ↓
3. 0.8秒后自动切换到下一个频道
   ↓
4. 从步骤1开始，测试新频道
```

## 📦 安装说明

### 系统要求
- **最低版本**: iOS 16.0
- **推荐版本**: iOS 17.0+
- **设备**: iPhone、iPad

### 安装方式

**方式1：使用 GitHub Actions 自动构建**
```bash
git tag -a v1.4.7-ios -m "iOS v1.4.7 Release"
git push origin v1.4.7-ios
```
等待 GitHub Actions 自动构建并生成 IPA 文件

**方式2：本地编译（需要 macOS + Xcode）**
```bash
cd TVPlayer-iOS
xcodegen generate
open TVPlayer.xcodeproj
# 在 Xcode 中选择 Product → Archive
```

**方式3：侧载安装**
1. 下载 IPA 文件
2. 使用 Sideloadly 或 AltStore 签名安装
3. 免费 Apple ID 每7天需重签

## ⚠️ 重要说明

### 关于之前版本的问题

**v1.4.6 及之前版本的问题：**
- 超时时间过长（4秒起播超时）
- 黑屏检测不够灵敏（3.5秒）
- 所有线路失败后切换太慢（1.5秒延迟）
- 单线路频道失败后不会自动切换

**v1.4.7 完全解决了这些问题！**

### 使用建议

1. **首次使用**: 启动后会自动加载频道列表
2. **遇到黑屏**: 不要手动切换，等待1.5-3秒自动切换
3. **频道卡顿**: 会自动检测并切换线路
4. **没有画面**: 系统会自动测试所有线路，然后切换频道
